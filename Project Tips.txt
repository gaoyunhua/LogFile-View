Project Requirements (Project Tips.txt)

1. Platform
   - Python 3.8+
   - PyQt5
   - Windows-based desktop application
   - SQLite database for persistence

2. Purpose
   - Parse and analyze log files by extracting structured data using regex-defined fields

3. Log File Handling
   - Default: reopen the last opened file on startup
   - UTF-8 encoding support (with error tolerance)
   - Handles random data and malformed lines gracefully
   - File is treated as line-separated text

4. UI Layout
   - Menu bar (File, Tools, Field Settings)
   - Ribbon-style toolbar with icons and tooltips
   - Main table view with raw log + extracted field columns

5. Code Organization
   - MVC architecture (models/, views/, controllers/)
   - Class-based code
   - Target file size limit: ~200 lines / ~1000 words per file
   - Files separated by purpose into directories

6. Data Display
   - Table view with sortable, filterable columns
   - Filter by field value (numeric or string range)
   - Field visibility is user-controllable
   - "Raw Log" column is always present (first column, minimal width)
   - Double-click column header to open field editor
   - Column width adjustable by dragging
   - Column position adjustable by drag-and-drop
   - Multi-line text wrapping in headers and cells

7. Field Editor
   - Define fields via regular expression
   - Fields can target fixed positions or keywords in a log line
   - Includes a "Test Regex" button to validate patterns
   - Field definitions are saved/loaded from SQLite database

8. Field List Dialog
   - Accessible from menu and toolbar
   - Displays: Field Name, Visibility, Sorted, Regular Expression
   - Click to edit any field definition

9. Regex Tester
   - Enter regex pattern and sample text
   - Displays extracted match result

10. Data Export
    - Export visible table data to Excel (.xlsx)
    - Copy selected cells to clipboard

11. Configuration Storage (SQLite)
    - Field definitions (name, regex, visibility, sorted, display order, column width)
    - Filter settings (field, low/high limits, type)
    - Last opened log file path
    - Window geometry (position and size)
