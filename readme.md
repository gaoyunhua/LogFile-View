# Log File Viewer

## Overview
The Log File Viewer is a Python-based application for analyzing and managing log files. It provides features such as:
- Viewing and filtering log data
- Managing field definitions with regular expressions
- Testing regular expressions
- Sorting and saving log data

## Requirements
- Python 3.8+
- PyQt5
- SQLite
